# Product Overview

## React Native Multi-TV App Sample

A production-ready TV application template for building cross-platform TV apps with a single React Native codebase.

### Supported Platforms
- **Android TV** - Google's TV platform
- **Apple TV** - tvOS devices
- **Fire TV (Fire OS)** - Amazon Fire TV with Fire OS
- **Fire TV (Vega OS)** - Amazon Fire TV with Vega OS (optimized build)
- **Web TV** - Browser-based TV platforms (LG WebOS, Samsung Tizen)

### Core Features
- **Video Playback** - Integrated video player with custom controls and remote support
- **Spatial Navigation** - TV-optimized focus management for D-pad/remote control
- **Dynamic Content** - External catalog API for loading movie metadata
- **Drawer Navigation** - Left-side menu with customizable items
- **Responsive Layouts** - Grid layouts optimized for TV screens (10-foot UI)
- **Platform-Specific Optimizations** - Automatic platform detection and optimized rendering

### App Variants
1. **expo-multi-tv** - Universal app supporting Android TV, Apple TV, Fire TV (Fire OS), and Web
2. **vega** - Fire TV Vega OS optimized build using Amazon Vega SDK

### Key Use Cases
- Streaming video applications
- Content discovery and browsing
- TV-optimized user interfaces
- Cross-platform TV development reference
